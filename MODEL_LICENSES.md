# Model and Checkpoint Licensing

The VisionForge Community Edition repository license covers first-party repository
source code only.

Models, checkpoints, weights, fonts, datasets, and other non-code assets are not
automatically covered by the repository Community Edition license.

## SAM 2 assets

If you enable SAM 2 support, you are responsible for obtaining, storing, and
using SAM 2 checkpoints and related assets under the applicable upstream terms.

Do not assume that:

- files under `/app/models`
- downloaded checkpoints
- upstream demo assets
- upstream fonts
- optional upstream native or helper code

are covered by the VisionForge Community Edition license.

## Customer-provided assets

Customer-provided models, checkpoints, images, videos, prompts, annotations, and
exports remain subject to the customer's own rights and upstream obligations.

## Distribution rule

If you ship containers, installers, or other binary deliverables that include
model files or checkpoints, include the required upstream notices and verify that
your redistribution path is allowed by the applicable upstream terms.
